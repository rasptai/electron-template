# React + TypeScript + Vite + Electron / Python + FastAPI

Web系技術を利用してデスクトップアプリケーションを作成するテンプレートです。
PandasなどのPythonライブラリを用いてデータ処理を行うため、
バックエンドはPython + FastAPIで実装し、フロントエンドからAPI通信で実行させます。
データの受け渡しはUploadなどを使用せず、ファイルパスの受け渡しにて行います。
Embedded Pythonをインストール時に展開することでユーザーはPython環境の準備が不要で、環境を独立させます。

## 環境の準備
1. git clone
2. npm install
